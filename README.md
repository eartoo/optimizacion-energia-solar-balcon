# optimizacion-energia-solar-balcon
Script en JavaScript para calcular la rentabilidad y el ahorro de kits solares de balcón en España. Incluye datos actualizados de tarifas eléctricas de 2026. Proyecto vinculado a mejorkitsolares.com.
